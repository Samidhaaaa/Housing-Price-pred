House Price Prediction

## Project Overview
This project focuses on predicting house prices using machine learning. The dataset was first cleaned and explored using different visualization techniques to understand the relationship between house prices and various features. Two machine learning models were then built and compared to predict house prices.

## Objective
The objective of this project is to predict house prices based on different property features such as area, bedrooms, bathrooms, parking, furnishing status, air conditioning and other amenities.

## Dataset
- **Dataset:** Housing.csv
- The dataset contains information about different houses along with their selling prices and property features.

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Jupyter Notebook

## Project Workflow
1. Data Loading
2. Data Cleaning
3. Exploratory Data Analysis (EDA)
4. Data Visualization
5. Feature Encoding
6. Model Building
7. Model Evaluation
8. Prediction and Comparison

## Machine Learning Models
- Linear Regression
- Random Forest Regressor

## Results
- Linear Regression achieved an **R² score of approximately 0.65**.
- The average prediction error was around **₹9.7 lakhs**.
- Surprisingly, Linear Regression performed better than Random Forest Regressor on this dataset.

## Key Insights
- Area is one of the most important factors affecting house prices.
- Houses with air conditioning and parking facilities generally have higher prices.
- Larger houses tend to have higher market values.
